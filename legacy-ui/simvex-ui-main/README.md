# Legacy UI 실행 안내

기존 Create React App 기반 UI 실행 문서.

## 실행 명령
- 개발 서버: `npm start`
- 테스트: `npm test`
- 프로덕션 빌드: `npm run build`
- eject: `npm run eject`

## 실행 경로
```bash
cd legacy-ui/simvex-ui-main
npm install
npm start
```

## 참고
- 개발 서버 기본 주소: `http://localhost:3000`
- API 프록시 대상: `http://localhost:8080`
- `eject` 실행 후 되돌리기 불가.
- 상세 문서: https://facebook.github.io/create-react-app/docs/getting-started
